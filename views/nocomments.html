I don't enjoy seeing the comments on most websites.
So I turned them off. You can too.

Want a site added to the list? Let me know.

Firefox

  1. Save 
    nocomments.css
     to your computer
  2. Find your Firefox profile: in Firefox, choose Help > Troubleshooting Information
  3. Under Application Basics, choose Show Folder
  4. Open the 
    chrome/
    folder (create it if it doesn't exist)
  5. Copy 
    nocomments.css
    to 
    chrome/
    as 
    userContent.css
  6. Restart Firefox

Safari

  1. Save 
    nocomments.css
     to your computer
  2. In Safari, choose Safari > Preferences > Advanced
  3. Under Style sheet select Other... and choose 
    nocomments.css
  4. Restart Safari

Internet Explorer

  1. Save 
    nocomments.css
     to your computer
  2. In Internet Explorer, choose the Tools icon > Internet options > General > Accessibility
  3. Check the Format documents using my style sheet box
  4. Browse and choose 
    nocomments.css
  5. Restart Internet Explorer

Chrome

  1. Save 
    nocomments.css
     to your computer
  2. Install the Stylish extension
  3. Click the new Stylish button > Manage installed styles > Write new style
  4. Open 
    nocomments.css
    and copy the entire contents to clipboard
  5. Paste contents to the Code area in Stylish
  6. Save new style
